GED — Gestion Électronique de Documents

Application de gestion documentaire permettant de stocker, organiser et partager des fichiers avec des droits d’accès simples. Objectif : centraliser les documents pour une petite équipe sans dépendre d’un service cloud tiers. Projet réalisé en stage chez Oxycad (Wiwersheim) — Janvier–Février 2026.

  • Stack : PHP, MySQL, HTML/CSS, un peu de JavaScript pour les interactions.
  • Fonctionnalités : upload, arborescence basique, recherche, rôles/utilisateurs.
  • Sécurité : 2FA, hashage + salage des mots de passe, contrôles d’accès renforcés.
  • En cours : finition UI et amélioration des contrôles d’accès.
